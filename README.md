# Skillcrush 104 Hangman

1. Clone the repo:
    ```
    git clone https://github.com/AbbiAld/hangman.git
    ```
2. Install all dependencies
    ```
    bundle install
    ```
3. Play!!

    On mac:
    ```
    bundle exec exe/hangman
    ```
    On Windows:
    ``` 
    bundle exec ruby exe/hangman
    ```
